
# Freelance Web Development Website

A fast, stylish multi-page static site for freelance web developers. Built with **HTML + CSS + JavaScript** (no build step), deployable to **GitHub Pages**, **Netlify**, or **Vercel**.

## Features
- ✅ Home with Hero, Core Value/Benefits, Featured Work, Trust Signals, and Final CTA
- ✅ Services (Starter Website, Custom Web App, E‑commerce) + 4‑step process (Discovery → Design → Development → Launch)
- ✅ Portfolio grid with links to per-project **Case Studies**
- ✅ Case Study template covering Challenge → Solution (Tech Stack) → Results → Links (Live + GitHub)
- ✅ About Me (Personal Story, Skill Stack, Mission)
- ✅ Contact (Form + direct Email/Phone/LinkedIn)
- ✅ Mobile-first, accessible, and performance‑minded design
- ✅ Version controlled and ready for free hosting

## Quick Start
1. **Edit config** in `site.config.json` (business name, CTA URL, email/phone, LinkedIn).
2. (Optional) Update project content in `assets/app.js` → `window.SITE_PROJECTS`.
3. Open `index.html` locally to preview (double‑click it).

## Deploy — GitHub Pages
1. Create a new GitHub repository (public).
2. Upload all files at the repo root (including `index.html`).
3. In **Settings → Pages**, set **Branch: main / root**. After it builds, your site goes live at a `github.io` URL.

## Deploy — Netlify
1. Go to [Netlify](https://app.netlify.com), click **New site from Git** and connect your repo.
2. No build command needed; publish directory = `/`.
3. Forms are auto‑enabled by Netlify via `data-netlify="true"`.

## Deploy — Vercel
1. Import your GitHub repo at [Vercel](https://vercel.com/new).
2. Framework preset = **Other**; Output Directory = `/`.
3. Deploy. Done!

## Customize
- Replace the three placeholder thumbnails in `/assets/img/` with your own images (keep file names or update `assets/app.js`).
- Update testimonials, logo strip, and copy in HTML if desired.
- For Calendly: set `primaryCTA.url` in `site.config.json` to your booking link.
